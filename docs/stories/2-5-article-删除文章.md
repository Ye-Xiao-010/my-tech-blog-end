# 删除文章 - 故事2.5

## 状态

- **当前状态**: done

## 故事

作为 博客作者,
我希望 能够删除自己的文章,
以便 管理和清理自己的内容。

## 验收标准

1. 作者可以删除自己的文章
2. 删除文章前应该显示确认提示
3. 删除文章后应该从列表中移除
4. 删除文章应该同时删除相关评论
5. 删除操作应该在1秒内完成
6. 系统应该记录文章删除日志
7. 只有文章作者可以删除文章
