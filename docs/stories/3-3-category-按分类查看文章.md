# 故事 3.3: 按分类查看文章

状态: implemented

## 故事

作为访客或已登录用户,
我希望能够按分类查看文章,
以便找到特定主题的内容。

## 验收标准

1. **Given** 我是访客或已登录用户
   **When** 我点击一个分类
   **Then** 我应该看到该分类下的所有文章
2. **And** 文章应该按最新排序
3. **And** 页面应该显示分类名称和描述
