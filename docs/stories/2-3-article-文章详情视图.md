# 文章详情视图 - 故事2.3

## 状态

- **当前状态**: completed

## 故事

作为 博客读者,
我希望 能够查看文章的详细内容,
以便 阅读完整的文章并获取所需信息。

## 验收标准

1. 读者可以查看单篇文章的完整内容
2. 文章详情应该显示文章标题、作者、发布时间、分类、标签和内容
3. 文章内容应该正确渲染富文本格式
4. 文章详情应该包含评论区
5. 文章详情应该包含相关文章推荐
6. 文章详情加载时间应该在2秒内完成
7. 文章详情应该在不同设备上正常显示
