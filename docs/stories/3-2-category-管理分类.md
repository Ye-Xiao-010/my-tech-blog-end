# 故事 3.2: 管理分类

状态: drafted

## 故事

作为管理员,
我希望能够编辑和删除分类,
以便维护分类体系。

## 验收标准

1. **Given** 我是管理员
   **When** 我访问分类管理页面
   **Then** 我应该看到所有现有分类的列表
2. **And** 我应该能够编辑分类名称和描述
3. **And** 我应该能够删除分类
4. **And** 删除分类前应该显示确认对话框
5. **And** 分类更新或删除应该在2秒内完成

## 任务/子任务

- [ ] 实现分类列表功能 (AC: 1)
  - [ ] 开发分类列表API
  - [ ] 实现分类列表UI
  - [ ] 添加分类排序和分页
- [ ] 实现分类编辑功能 (AC: 2)
  - [ ] 开发分类编辑表单
  - [ ] 实现分类编辑API
  - [ ] 添加编辑权限验证
- [ ] 实现分类删除功能 (AC: 3, 4)
  - [ ] 开发分类删除API
  - [ ] 添加删除确认对话框
  - [ ] 实现分类删除前的关联检查
- [ ] 优化分类管理性能 (AC: 5)
